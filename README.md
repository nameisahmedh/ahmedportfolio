## ğŸƒ Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nameisahmedh/ahmedportfolio.git
   cd ahmedportfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)
